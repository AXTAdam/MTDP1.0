# Figures

`preview/` contains PNG versions of Figure 4 to Figure 8 for GitHub display.

`high_resolution/` currently contains `Figure4.tiff`, which is small enough for normal GitHub upload.

The original high-resolution TIFF files for Figure 5 to Figure 8 are larger than GitHub's standard 100 MB per-file limit. To publish those files, use one of the following approaches:

- Git LFS for storing large TIFF files in the repository.
- GitHub Releases for attaching high-resolution TIFF files as downloadable assets.
- An external data repository if the final journal or institution requires long-term archival storage.
