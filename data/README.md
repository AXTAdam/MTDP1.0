# Data

This folder contains public Excel tables derived from the music therapy knowledge database.

Included:

- Disease information
- Study population characteristics
- Reference information
- Study design information
- Music-based intervention design
- Music therapy-based technologies
- Session design
- Intervention music design
- Standardized summary table

Not included:

- `SummaryofKnowledgeBaseData.xlsx`, the original unstandardized master table.

The standardized summary table reflects explicit abbreviation and terminology mappings. Values without an explicit mapping rule were retained as originally recorded.
